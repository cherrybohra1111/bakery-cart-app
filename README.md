# Bakery Shop

A simple bakery cart application built with HTML, CSS, and JavaScript.

## Features

-  Display bakery products
-  Add items to cart
-  Merge duplicate items with quantity
-  Increase quantity
-  Decrease quantity
-  Remove item
-  Calculate total cost
-  Checkout and clear cart

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## What I Learned

- DOM manipulation
- Event delegation
- Dynamic rendering
- Array methods (find, findIndex, forEach)
- State management with JavaScript objects
